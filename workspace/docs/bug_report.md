# 【至急】6月分の売上レポートが出力できない

**報告者**: 営業事務・佐藤
**日付**: 2026-07-01
**優先度**: 高（月初の経営会議で6月レポートが必要）

## 現象

毎月使っている売上レポートツールで、6月分だけエラーになります。
5月分までは問題なく出力できていました。

実行したコマンド:

```
python3 -m report_tool.cli data/sales_2026.csv --month 2026-06
```

エラーの最後の部分:

```
  File "report_tool/aggregate.py", line 23, in category_summary
    "平均単価": sales[cat] / qty[cat],
ZeroDivisionError: division by zero
```

## 心当たり

6月は展示会があり、ノベルティ（サンプル品）の配布記録をデータに
追加しました。売上にはならないので数量0で登録しています。
それが関係しているかもしれません。

サンプル品の行を消せば動くのかもしれませんが、配布記録として
残したいので、データを消さずにツール側で対応してほしいです。

## P.S.

経理からは以前より「返品があった月も売上が減っていないように見える」
と言われています。こちらは急ぎませんが、時間があるときに確認して
もらえると助かります。
